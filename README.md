# BOIN-CX

Bayesian Optimal INterval Design Extensions for Drug-Combination Phase I Clinical Trials (BOIN-CX)

### Instructions to download and install

#Install devtools if not installed

`install.packages("devtools")`

#Install package BOIN.CX

`devtools::install_github("chnyuxuan/BOIN-CX")`

#To browse the package functions, type

`help(package="BOIN.CX")`
